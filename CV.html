Transportation Order Management

This project addresses the problem of transportation by allowing users to create groups or join existing ones. Once the group is complete and the time and destination are set, drivers can submit their offers for the group. Users can then choose the best offer for them. Once the supervisor approves the group, the transportation process begins. The front-end interfaces were designed using HTML, CSS, JS, and Bootstrap while the back-end was programmed using PHP and MySQL.
